# Library Book Management System

This project implements a simple Library Book Management System using Python.
The system is developed using a three-sprint Agile process and Git for version control.
All data is stored in memory using Python data structures (no database).

## Features
- Add books with unique Book IDs
- Borrow and return books
- Generate a library status report
- Unit testing using Python unittest framework

## Technology Used
- Programming Language: Python
- Testing Framework: unittest
- Version Control: Git and GitHub

## How to Run Tests
```bash
python -m unittest discover -s tests -p "test_*.py" -v
